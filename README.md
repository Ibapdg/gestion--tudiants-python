# Gestion des étudiants en Python 
Programme permettant de gérer une liste d'etudiant
## Fonctionnalités 
Le programme permet de : 
_ Ajouter un étudiant
_ Afficher la liste des étudiants 
_ Supprimer un étudiant 
_ Quitter le programme
## Comment fonctionne le programme 
Lorsque le programme est lancé, un menu apparait :
1. Ajouter un étudiant
2. Afficher les étudiants
3. Supprimer un étudiant
4. Quitter
